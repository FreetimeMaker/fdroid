# F-Droid Repository by Freetime Maker

The Official F-Droid Repository from Freetime Maker featuring open-source Android applications.

## 📱 Available Apps

### GeoWeather
- **Package ID:** `com.freetime.geoweather`
- **Description:** Modern weather app with 7-day forecast for multiple cities
- **Category:** Weather

### SSMP Controller
- **Package ID:** `com.freetime.ssmpc`
- **Description:** The Unofficial Companion App for the SuperSMP Minecraft Server.
- **Category:** Games, Internet

## 🔧 Repository Setup

To add this repository to your F-Droid client:

1. Open F-Droid
2. Go to Settings → Repositories
3. Add new repository with URL: `https://raw.githubusercontent.com/FreetimeMaker/fdroid/main/`
4. Scan QR code or enter repository details

## 📋 Requirements

- Android 7.0+ (API level 24+)
- F-Droid client application
- Internet connection for repository updates

## 🛠️ Development

This repository uses the standard F-Droid repository structure:

- `metadata/` - App metadata and build configurations
- `repo/` - Built APK files and repository index
- `archive/` - Older versions of applications
- `srclibs/` - Source libraries
